# University Course Management System

A relational database system designed and implemented for managing university academic operations. The system handles departments, instructors, students, courses, course sections, enrollments, classrooms, and examinations.

## Features
- Department and instructor management
- Student records and academic advisors
- Course and section management
- Student enrollment and grade tracking
- Examination management and results
- Relational database design with integrity constraints

## Database Concepts Applied
- Entity-Relationship Diagram (ERD)
- Relational Schema Mapping
- Primary and Foreign Keys
- Many-to-Many Relationships
- Check Constraints
- Data Integrity Rules
- Oracle SQL and PL/SQL

## Technologies Used
- Oracle Database
- SQL
- PL/SQL
- Database Design and Modeling
